# 🏥 Спортивное врачевание

Интерактивный тренажёр для врачей спортивных команд, ортопедов и реабилитологов.

## 🎯 Цели проекта

1. **Образовательная** — предоставить врачам и студентам возможность практиковаться в диагностике и лечении типичных спортивных травм и состояний.
2. **Интерактивная проверка знаний** — система баллов и статусов «решено/не решено» мотивирует пользователя проходить кейсы.
3. **Практическая направленность** — все кейсы основаны на реальных ситуациях из практики спортивных врачей.

## 🛠️ Технологии

- HTML5, CSS3 (модульная сетка 8px, адаптивность)
- JavaScript (ES6+, хранение данных в localStorage)

## 📁 Структура

- `index.html` — страница входа/регистрации
- `tasks.html` — список кейсов спортивной медицины с фильтрацией
- `detail.html` — детальный просмотр кейса
- `common.js` — база кейсов и логика работы с пользователями
- `style.css` — стили (модульная сетка, акценты, адаптивность)

## 📌 Как запустить

1. Скачайте все файлы в одну папку.
2. Откройте `index.html` в браузере.
3. Войдите под демо-логином `doctor` / пароль `123`.

## 🔒 Хранение данных

Прогресс пользователя (решённые кейсы, набранные баллы) сохраняется в `localStorage` браузера.

## 🧩 Кейсы спортивной медицины (6)

- **Синдром перетренированности** (кардиология)
- **Растяжение связок голеностопа** (травматология)
- **Тепловой удар у спортсмена** (неотложная помощь)
- **Сотрясение мозга в спорте** (травматология)
- **Внезапная сердечная смерть** (неотложная помощь)
- **Обезвоживание и электролитный дисбаланс** (неотложная помощь)
