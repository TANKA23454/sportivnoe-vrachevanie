# 🏥 Спортивное врачевание

Интерактивный веб-тренажёр для спортивных врачей, ортопедов и реабилитологов.

## 🎯 Цели проекта

1. **Образовательная** — предоставить врачам и студентам возможность практиковаться в диагностике и лечении типичных спортивных травм и состояний.
2. **Интерактивная проверка знаний** — система баллов и статусов «решено/не решено» мотивирует пользователя проходить кейсы.
3. **Практическая направленность** — все кейсы основаны на реальных ситуациях из практики спортивных врачей.

## 🛠️ Технологии

- HTML5, CSS3 (модульная сетка 8px, адаптивность)
- JavaScript (ES6+, хранение данных в localStorage)

## 📁 Структура

- `index.html` — главная страница со списком кейсов, фильтрами, прогрессом.
- `style.css` — все стили проекта (подключены к index.html).

## 📌 Как запустить

1. Скачайте все файлы (`index.html`, `style.css`) в одну папку.
2. Откройте `index.html` в браузере.

## 🔒 Хранение данных

Прогресс пользователя (решённые задачи, набранные баллы) сохраняется в `localStorage` браузера.

## 🧩 Кейсы спортивной медицины

Проект включает 3 сценария:
- **Синдром перетренированности** (кардиология)
- **Растяжение связок голеностопа** (травматология)
- **Тепловой удар у спортсмена** (неотложная помощь)

## ⚙️ Возможности для доработки

- Добавление новых кейсов в массив `TASKS` (файл `index.html`, секция JavaScript).
- Подключение бэкенда для совместного использования и сохранения прогресса между устройствами.
- Расширение детальной карточки задачи с выбором ответа и таймером.
